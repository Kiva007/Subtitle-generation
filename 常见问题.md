# 常见问题解答

## 1. HF Token 警告信息

**警告内容：**
```
Warning: You are sending unauthenticated requests to the HF Hub. Please set a HF_TOKEN to enable higher rate limits and faster downloads.
```

**这是什么意思？**
这个警告来自 Hugging Face Hub，表示在使用 Transformers 库时没有提供认证令牌（HF_TOKEN）。

**是否影响功能？**
- **不影响本地模型**：如果你已经下载并使用了本地模型（如 kotoba-tech/kotoba-whisper-v2.1），这个警告不会影响程序运行。
- **仅影响新模型下载**：如果需要下载新的模型或配置文件，没有 HF_TOKEN 会受到速率限制，下载速度可能较慢。

**如何解决（可选）？**
如果你需要下载新模型，可以按以下步骤设置 HF_TOKEN：

1. 访问 [Hugging Face Tokens](https://huggingface.co/settings/tokens)
2. 创建一个新的 Access Token（选择 "Read" 权限即可）
3. 设置环境变量：
   ```bash
   # Windows CMD
   set HF_TOKEN=your_token_here

   # Windows PowerShell
   $env:HF_TOKEN="your_token_here"

   # Linux/Mac
   export HF_TOKEN="your_token_here"
   ```

**结论：** 如果只使用已下载的本地模型，可以忽略此警告。

## 2. 日志中的日语乱码问题

**问题现象：**
处理日志中显示的日语文字变成乱码。

**解决方案：**
已在最新版本中修复，通过以下方式确保 UTF-8 编码：
- 子进程输出使用 UTF-8 编码
- GUI 正确处理 UTF-8 字符
- Windows 环境变量设置 PYTHONIOENCODING=utf-8

## 3. 进度条更新问题

**问题现象：**
进度条只在完成后从 0% 跳到 100%，中间没有实时更新。

**解决方案：**
已在最新版本中修复，进度条现在会：
- 根据 `[当前/总数]` 格式实时更新
- 转录阶段：20%-100%
- 显示当前处理进度

## 4. DLL 加载错误

**问题现象：**
```
[WinError 1114] 动态链接库(DLL)初始化例程失败
```

**解决方案：**
采用子进程架构，PyTorch 在独立进程中运行，避免与 PyQt6 的 DLL 冲突。已在 v1.1 版本中修复。
