## 设计目标

优化字幕生成器的 GUI 界面，主要关注布局调整和信息层次优化，保持现有功能代码运行逻辑不变。

---

## 设计方案

### 1. 布局结构

采用**左侧可收缩面板 + 右侧自适应工作区**的分栏布局：

- **左侧设置面板**（320px 宽度，可收缩）
  - AI 模型配置
  - 输出格式设置
  - 输出目录设置
  - 收缩功能：完全隐藏，宽度变为 0

- **右侧主要工作区**（自适应宽度）
  - 文件队列管理
  - 控制按钮组
  - 进度显示区域
  - 日志输出区域

### 2. 视觉风格

**主题**: 深色专业技术风格
**配色方案**: 蓝色工业风

#### 色彩规范

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主色调 | `#60a5fa` | 蓝色，用于按钮、链接、重点信息 |
| 次要色 | `#3b82f6` | 深蓝色，用于渐变、悬停状态 |
| 背景渐变 | `#1e3a5f → #0f172a` | 深蓝渐变背景 |
| 面板背景 | `#0f172a` | 深色面板背景 |
| 边框颜色 | `#334155` | 灰蓝色边框 |
| 主要文字 | `#e0e6ed` | 白灰色主要文字 |
| 次要文字 | `#94a3b8` | 灰色次要文字 |
| 成功状态 | `#22c55e` | 绿色 |
| 警告状态 | `#fbbf24` | 黄色 |
| 错误状态 | `#ef4444` | 红色 |

### 3. 字体方案

- **技术信息**: 等宽字体 (`Consolas`, `Monaco`, `Courier New`)
- **其他内容**: 系统默认字体 (`Microsoft YaHei`, `微软雅黑`, `Arial`)

**应用场景**:
- 等宽字体：模型 ID、URL 地址、文件大小、进度百分比、日志内容、时间戳
- 系统字体：标题、按钮文字、标签、描述文字、状态文字

### 4. 组件设计

#### 4.1 左侧设置面板

**AI 模型配置区域**:
- Whisper 模型输入框（等宽字体）
- 翻译模型下拉选择框
- LM Studio 地址输入框（等宽字体）
- 批量大小数值输入框（等宽字体，7-150 范围）

**输出格式区域**:
- 复选框组：原文字幕、中文字幕、双语字幕
- 复选框组：过滤语气词、调试模式

**输出设置区域**:
- 输出目录输入框（等宽字体）
- 复选框组：保留临时文件、递归扫描文件夹

#### 4.2 右侧工作区

**文件队列区域**:
- 标题 + 操作按钮组（选择文件、添加文件夹、清空列表）
- 文件列表表格（6列：序号、文件名、大小、状态、进度、操作）
- 统计信息栏（总计、已完成、处理中、等待中、失败）

**控制按钮区域**:
- 4 个主要按钮：开始处理、暂停、停止、重试失败
- 主按钮使用渐变背景，次级按钮使用边框样式

**进度显示区域**:
- 总体进度条 + 百分比显示
- 当前文件进度条 + 状态文字
- 状态信息文字

**日志输出区域**:
- 深色背景日志框
- 等宽字体显示
- 不同日志级别颜色区分（信息、成功、注意、错误）
- 自动滚动显示最新日志

#### 4.3 收缩功能

- 收缩按钮位于标题"字幕生成器"左侧
- 点击收缩时，左侧面板完全隐藏（宽度变为 0）
- 按钮图标在 ◀ 和 ▶ 之间切换
- 平滑的过渡动画效果

### 5. 交互细节

#### 按钮交互
- 悬停时背景色变化
- 主按钮有轻微上移效果
- 点击时有视觉反馈

#### 输入框交互
- 聚焦时边框高亮
- 焦点光晕效果（蓝色半透明）
- 平滑的过渡动画

#### 表格交互
- 行悬停时背景色微变
- 状态标签使用胶囊样式
- 操作列显示删除图标

#### 进度条
- 渐变填充效果
- 柔和的发光阴影
- 平滑的宽度过渡动画

#### 滚动条
- 自定义滚动条样式
- 与主题色彩一致
- 悬停时高亮显示

### 6. 响应式设计

- **大屏幕** (>1200px): 左侧面板 320px
- **中等屏幕** (768px-1200px): 左侧面板 280px
- **小屏幕** (<768px): 垂直布局，控制按钮自动换行

## 技术实现要点

### PyQt6 实现要点

1. **布局管理器**
   - 使用 `QHBoxLayout` 实现左右分栏
   - 左侧面板使用 `QVBoxLayout`
   - 右侧工作区使用 `QVBoxLayout`
   - 使用 `QSplitter` 实现可调整宽度的分栏

2. **样式表 (QSS)**
   - 定义全局颜色变量
   - 组件级别的样式定制
   - 状态样式（hover, focus, disabled）

3. **字体设置**
   - 技术信息组件设置等宽字体
   - 其他组件使用系统默认字体
   - 使用 `QFont` 和 `QFontDatabase`

4. **动画效果**
   - 使用 `QPropertyAnimation` 实现平滑过渡
   - 面板收缩/展开动画
   - 进度条动画

5. **自定义组件**
   - 自定义进度条组件（带渐变和发光效果）
   - 自定义状态标签组件
   - 自定义日志显示组件

### 保持功能逻辑不变的原则

- **只修改界面代码**: `gui_main.py` 中的界面构建部分
- **不修改业务逻辑**: `BatchProcessor`, `TaskThread`, `subprocess_processor.py` 等保持不变
- **保持信号-槽连接**: 现有的信号槽机制保持不变
- **配置文件兼容**: `gui_config.ini` 结构保持兼容
